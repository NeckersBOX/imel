Get a line points.

This function get all points in the line from coordinate \((\_x_1,\_y_1)\) to coordinate \((\_x_2,\_y_2)\). You can get the line width and height respectively from lx and ly argument.

Parameters
_x1Start line x coordinate
_y1Start line y coordinate
_x2End line x coordinate
_y2End line y coordinate
lxNULL or if passed, this function put inside it the line width
lyNULL or if passed, this function put inside it the line height
value_typeType of value passed as value
valueNumber of points you want to get from the line. The points will be uniformely distributed.
Returns
A NULL-terminated array with the line points.
