typedef struct _imel_error ImelError

Specialized type in reporting errors inside Imel.

This type report an error if occurred when doing operations in Imel. Each error have a code and a description.

Note
Usually it can be used in this way:
1 ImelImage *image;
2 ImelError error;
3 
4 image = imel_image_new_from ("image.png", 0, &error);
5 if ( !image ) {
6  fprintf (stderr, "Error %d: %s\n", error.code, error.description);
7  return 1;
8 }

enum _imel_effect

Effects to apply through imel_image_apply_effect or some other function.

Note
Enum values starts from 0
See also
imel_image_apply_effect
Enumerator
IMEL_EFFECT_WHITE_BLACK 

Grayscale effect to an image.

IMEL_EFFECT_ANTIQUE 

Antique effect ( less bright colors ).

IMEL_EFFECT_INVERT 

Invert colors.

IMEL_EFFECT_NORMALIZE 

Calculates the average of colors of the image and apply it to RGB channel selected. When used as a parameter for imel_image_apply_effect function takes as a parameter a value of ImelMask type (with the option to combine multiple channels with the OR operator).

IMEL_EFFECT_BRIGHTNESS 

Affect the image brightness. When used as a parameter for imel_image_apply_effect function takes as a parameter a value between -100 and +100.

IMEL_EFFECT_CONTRAST_STRETCHING 

Auto contrast enhancement.

IMEL_EFFECT_CONTRAST 

Affect the image contrast. When used as a parameter for imel_image_apply_effect function takes as a parameter a number between -127 and +128.

IMEL_EFFECT_RASTERIZE 

Rasterizes the image, which divides the image into squares of size chosen, colored with the average of pixels that are inside each square. When used as a parameter for the imel_image_apply_effect function it takes as a parameter the size in pixels of the side of the square.

IMEL_EFFECT_ANTIALIAS 

Apply antialiasing effect. First calculating the value that takes each pixel of the image and then sets it. When used as a parameter for imel_image_apply_effect function it takes as a parameter the number of pixels of the side of the square to be analyzed to calculate the value of the current pixel.

IMEL_EFFECT_DIRECT_ANTIALIAS 

As IMEL_EFFECT_ANTIALIAS except for the fact that the value of the current pixel applies it immediately after it is calculated.

IMEL_EFFECT_IMAGE_ADD 

Sum the colors of an image to the colors of another. When used as a parameter for the function imel_image_apply_effect requires as a parameter the image to be added.

IMEL_EFFECT_IMAGE_SUBTRACT 

Subtract the colors of an image to the colors of another. When used as a parameter for the function imel_image_apply_effect requires as a parameter the image to be subtracted.

IMEL_EFFECT_COLOR_TO_ALPHA 

It allows to bring a shade of color to transparency to creating the right shades so that the resulting image can be retrofit on a different background. It receives as a parameter the color to be eliminated in the form of pointer to ImelPixel variable.
